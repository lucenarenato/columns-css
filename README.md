> Descrição
> Neste direto ao ponto, mostraremos como organizar textos em colunas utilizando CSS com ajuda da propriedade Columns, que chegou para realizar essa função com zero esforço.

- css

```
<style>
		.container {
			width: 600px;
			margin: 0 auto;
		}
		.container p {
			columns: 2;
		}

	</style>
  ```
